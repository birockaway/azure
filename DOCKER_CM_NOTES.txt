Build image 
docker build --no-cache=true -t crossmasters-docker-test .

Pustit docker image bez spusteni aplikace, override entrypoin
 docker run -ti --entrypoint=//bin//bash crossmasters-docker-test

Pro testovaci pusteni musim odkazat na sdilenou slozku, by default Users, jinak prazdny mount
ex.: docker run -it --volume=//c/Users/David/data://data keboola/ftpex



docker run -it --volume=//c/Users/esner/Documents/Prace/KBC/CrossmasterDocker/data://data crossmasters-docker-test



-m="512m" limit ram
run -d :run in detached mode
docker images :list images
